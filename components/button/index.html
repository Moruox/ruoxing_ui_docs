RxButton | 基于Element Plus 封装 ​

常用的按钮组件。

TIP

新增

  • 重写@click事件，使得按钮触发逻辑自带截流防抖功能。
  • 新增属性 clickType 控制触发模式，可选值为 throttle 或 debounce。默认值为 debounce。
  • 新增属性 clickDelay 控制触发延迟，单位为 ms。默认值为 300。

基本用法 ​

使用 type, plain, round 和 circle 来定义按钮的样式。

<template>
  <div class="mb-4">
    <rx-button>Default</rx-button>
    <rx-button type="primary">Primary</rx-button>
    <rx-button type="success">Success</rx-button>
    <rx-button type="info">Info</rx-button>
    <rx-button type="warning">Warning</rx-button>
    <rx-button type="danger">Danger</rx-button>
  </div>

  <div class="mb-4">
    <rx-button plain>Plain</rx-button>
    <rx-button type="primary" plain>Primary</rx-button>
    <rx-button type="success" plain>Success</rx-button>
    <rx-button type="info" plain>Info</rx-button>
    <rx-button type="warning" plain>Warning</rx-button>
    <rx-button type="danger" plain>Danger</rx-button>
  </div>

  <div class="mb-4">
    <rx-button round>Round</rx-button>
    <rx-button type="primary" round>Primary</rx-button>
    <rx-button type="success" round>Success</rx-button>
    <rx-button type="info" round>Info</rx-button>
    <rx-button type="warning" round>Warning</rx-button>
    <rx-button type="danger" round>Danger</rx-button>
  </div>

  <div>
    <rx-button :icon="Search" circle />
    <rx-button type="primary" :icon="Edit" circle />
    <rx-button type="success" :icon="Check" circle />
    <rx-button type="info" :icon="Message" circle />
    <rx-button type="warning" :icon="Star" circle />
    <rx-button type="danger" :icon="Delete" circle />
  </div>
</template>

<script lang="ts" setup>
import {
  Check,
  Delete,
  Edit,
  Message,
  Search,
  Star,
} from '@element-plus/icons-vue'
</script>

禁用按钮 ​

disabled 属性决定按钮是否禁用。 使用 disabled 属性来决定按钮是否禁用。它接受一个 Boolean 值。

<template>
  <div class="mb-4">
    <rx-button disabled>Default</rx-button>
    <rx-button type="primary" disabled>Primary</rx-button>
    <rx-button type="success" disabled>Success</rx-button>
    <rx-button type="info" disabled>Info</rx-button>
    <rx-button type="warning" disabled>Warning</rx-button>
    <rx-button type="danger" disabled>Danger</rx-button>
  </div>
  <div>
    <rx-button plain disabled>Plain</rx-button>
    <rx-button type="primary" plain disabled>Primary</rx-button>
    <rx-button type="success" plain disabled>Success</rx-button>
    <rx-button type="info" plain disabled>Info</rx-button>
    <rx-button type="warning" plain disabled>Warning</rx-button>
    <rx-button type="danger" plain disabled>Danger</rx-button>
  </div>
</template>

链接按钮 ​

Basic link button

Disabled link button

<template>
  <p>Basic link button</p>
  <div class="mb-4">
    <rx-button
      v-for="button in buttons"
      :key="button.text"
      :type="button.type"
      link
    >
      {{ button.text }}
    </rx-button>
  </div>

  <p>Disabled link button</p>
  <div>
    <rx-button
      v-for="button in buttons"
      :key="button.text"
      :type="button.type"
      link
      disabled
    >
      {{ button.text }}
    </rx-button>
  </div>
</template>

<script setup lang="ts">
const buttons = [
  { type: '', text: 'plain' },
  { type: 'primary', text: 'primary' },
  { type: 'success', text: 'success' },
  { type: 'info', text: 'info' },
  { type: 'warning', text: 'warning' },
  { type: 'danger', text: 'danger' },
] as const
</script>

DANGER

警告
type="text" 已被 弃用，并将在 3.0.0 中被 移除，请考虑切换到新的 API。 新 API link 已在 2.2.1 中添加，您可以使用 type API 来设置链接按钮的主题。

图标按钮 ​

使用图标为按钮添加更多含义。您可以仅使用图标以节省空间，或者将图标与文本一起使用。

<template>
  <div>
    <rx-button type="primary" :icon="Edit" />
    <rx-button type="primary" :icon="Share" />
    <rx-button type="primary" :icon="Delete" />
    <rx-button type="primary" :icon="Search">Search</rx-button>
    <rx-button type="primary">
      Upload<rx-icon class="rx-icon--right"><Upload /></rx-icon>
    </rx-button>
  </div>
</template>
<script setup lang="ts">
import { Delete, Edit, Search, Share, Upload } from '@element-plus/icons-vue'
</script>

使用 icon 属性来添加图标。您可以在 Element Plus 图标组件中找到图标列表。通过 <i> 标签将图标添加到文本的右侧是可行的。也可以使用自定义图标。

按钮组 ​

以按钮组的形式显示，可用于对一系列相似操作进行分组。 使用 <el-button-group> 标签来对按钮进行分组。

Previous Page Next Page
<template>
  <el-button-group>
    <rx-button type="primary" :icon="ArrowLeft">Previous Page</rx-button>
    <rx-button type="primary">
      Next Page<el-icon class="rx-icon--right"><ArrowRight /></el-icon>
    </rx-button>
  </el-button-group>
  <el-button-group class="ml-4">
    <rx-button type="primary" :icon="Edit" />
    <rx-button type="primary" :icon="Share" />
    <rx-button type="primary" :icon="Delete" />
  </el-button-group>
</template>

<script setup lang="ts">
import {
  ArrowLeft,
  ArrowRight,
  Delete,
  Edit,
  Share,
} from '@element-plus/icons-vue'
</script>

加载按钮 ​

点击按钮加载数据后，按钮显示加载状态。

将 loading 属性设置为 true 以显示加载状态。

<template>
  <rx-button type="primary" loading>Loading</rx-button>
  <rx-button type="primary" :loading-icon="Eleme" loading>Loading</rx-button>
  <rx-button type="primary" loading>
    <template #loading>
      <div class="custom-loading">
        <svg class="circular" viewBox="-10, -10, 50, 50">
          <path
            class="path"
            d="
            M 30 15
            L 28 17
            M 25.61 25.61
            A 15 15, 0, 0, 1, 15 30
            A 15 15, 0, 1, 1, 27.99 7.5
            L 15 15
          "
            style="stroke-width: 4px; fill: rgba(0, 0, 0, 0)"
          />
        </svg>
      </div>
    </template>
    Loading
  </rx-button>
</template>

<script lang="ts" setup>
import { Eleme } from '@element-plus/icons-vue'
</script>

<style scoped>
.rx-button .custom-loading .circular {
  margin-right: 6px;
  width: 18px;
  height: 18px;
  animation: loading-rotate 2s linear infinite;
}
.rx-button .custom-loading .circular .path {
  animation: loading-dash 1.5s ease-in-out infinite;
  stroke-dasharray: 90, 150;
  stroke-dashoffset: 0;
  stroke-width: 2;
  stroke: var(--rx-button-text-color);
  stroke-linecap: round;
}
</style>

提示
您可以使用 loading 插槽或 loadingIcon 来自定义您的加载组件。
ps: loading 插槽的优先级高于 loadingIcon

尺寸 ​

除了默认尺寸外，按钮组件还为您提供了三种额外的尺寸供您在不同场景中选择。 使用 size 属性来设置 large, small 等额外尺寸。

<template>
  <div class="flex flex-wrap items-center mb-4">
    <rx-button size="large">Large</rx-button>
    <rx-button>Default</rx-button>
    <rx-button size="small">Small</rx-button>
    <rx-button size="large" :icon="Search">Search</rx-button>
    <rx-button :icon="Search">Search</rx-button>
    <rx-button size="small" :icon="Search">Search</rx-button>
  </div>
  <div class="flex flex-wrap items-center mb-4">
    <rx-button size="large" round>Large</rx-button>
    <rx-button round>Default</rx-button>
    <rx-button size="small" round>Small</rx-button>
    <rx-button size="large" :icon="Search" round>Search</rx-button>
    <rx-button :icon="Search" round>Search</rx-button>
    <rx-button size="small" :icon="Search" round>Search</rx-button>
  </div>
  <div class="flex flex-wrap items-center">
    <rx-button :icon="Search" size="large" circle />
    <rx-button :icon="Search" circle />
    <rx-button :icon="Search" size="small" circle />
  </div>
</template>

<script setup lang="ts">
import { Search } from '@element-plus/icons-vue'
</script>

自定义颜色 beta ​

您可以自定义按钮颜色。

<template>
  <div class="mb-4">
    <rx-button v-for="color in colors" :key="color" :color="color">{{ color }}</rx-button>
  </div>
</template>

<script lang="ts" setup>
const colors = ['#4CAF50', '#2196F3', '#FF5722', '#9C27B0', '#E91E63']
</script>

提示
我们将自动计算悬停颜色和活动颜色。
