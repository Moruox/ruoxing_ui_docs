Form 表单

目录

RxForm | 基于 Element Plus 封装 ​

表单套件合集，提供高效的表单开发体验。

特性说明 ​

变更信息

  • 精简模版语法：以配置项快速生成表单结构。
  • 布局组件：内置 row 和 col，通过 span 属性控制表单布局。
  • 内置校验：支持 model 和基础必填 rules 快速完成表单校验。
  • 动态表单：支持根据数据变化动态生成表单及后端数据结构。
  • 灵活渲染：支持内嵌组件渲染，兼容 vue 组件及 jsx/tsx 语法。

基本用法 ​

使用 form-items 和 form-props 快速生成表单。

<template>
  <div class="mb-4">
    <rx-form
        :formProps="formProps"
        :form-items="formItems"
    />
  </div>
</template>
<script lang="ts" setup>
//内置的生成响应式表单的工具函数,返回一个代理对象,为了更好的支持TS请使用该函数生成配置项
import { useFormReactive , useFormItemReactive} from "@ruoxing/ruoxing-ui"
// 对应el-form的props 如label-width等
const formProps = useFormReactive({
  labelWidth: '40px',
  //新增属性(控制每一个表单项/el-form-item的占位宽度)
  span: 24,
})
// 表单项/el-form-item的生成配置
const formItems = useFormItemReactive([
  {
    // 对应el-form-item的props 如label等
    formItemProps:{
      label: "名称",
      prop: "username",
    },
    //想要在el-form-item中渲染的组件名(el-input)/.vue文件/tsx语法
    component:'el-input',
    //需要传递给组件的props
    props:{
      //默认的v-model绑定值
      vModel:'',
      placeholder: "请输入用户名"
    }
  },
  {
    formItemProps:{
      label: "密码",
      prop: "password",
    },
    component:'el-input',
    props:{
      type: 'password',
      vModel:'',
      placeholder: "请输入密码",
      type: 'password'
    }
  },
])
</script>

数据操作 ​

获取表单数据 ​

监听 data-change 事件，表单数据变化时自动生成需要返回给后端的数据结构。

生成的表单数据：{}
<template>

通过 useInsertFormItemReactiveData 插入数据到 form-items。

        :form-items="formItems"

高级功能 ​

条件渲染与数据源配置 ​

  • 使用 form-items 中的 formItemProps.hidden: true 控制表单项显示/隐藏。
  • 修改 form-props 时推荐使用 useSetReactiveData。
<template>
  <div class="mb-4">
    <rx-form
        ref="rxFormRef"

        :form-items="formItems"

<script lang="ts" setup>
//内置的生成响应式表单的工具函数,返回一个代理对象,为了更好的支持TS请使用该函数生成配置项
import { useFormReactive , useFormItemReactive , useSetReactiveData} from "@ruoxing/ruoxing-ui"
import {reactive, ref} from "vue";
const rxFormRef = ref()
const formData = reactive({})
// 对应el-form的props 如label-width等
const formProps = useFormReactive({
  labelWidth: '70px',
  labelPosition: 'left',

// 表单项/el-form-item的生成配置
const formItems = useFormItemReactive([
  {

    formItemProps:{
      label: "名称",
      prop: "username",
    },
    //想要在el-form-item中渲染的组件名(el-input)/.vue文件/tsx语法
    component:'el-input',
    //需要传递给组件的props
    props:{
      //默认的v-model绑定值
      vModel:'',
      placeholder: "请输入用户名",
    }
  },
  {
    formItemProps:{
      label: "密码",
      prop: "password",
    },
    component:'el-input',
    props:{
      type: 'password',
      vModel:'',
      placeholder: "请输入密码",
      type: 'password'
    }
  },
  {
    formItemProps:{
      label: "邮箱",
      prop: "email",
      hidden:true,
    },
    component:'el-input',
    props:{
      vModel:'',
      placeholder: "请输入邮箱",
    }
  },
])
const renderEmailOption = () => {
  /**调用插入方法
   * @param formItems 表单项formItems的生成配置或者formProps对象
   * @param key 要设置成员的key访问路径
   * (支持语法 prop:keyPath(通过.语法访问对象)
   * 或者 prop:keyName(仅支持keyName在当前对象中唯一时使用))
   * 单设置对象为formProps时，prop:可以省略，直接写keyPath即可
   * @param value 需要插入的数据源
   */
  useSetReactiveData(formItems,'email:formItemProps.hidden',false)
}
</script>

表单校验 ​

  • 使用 form-props.allRequired: true 启用所有项必填校验。
  • 可以在 form-items 项中配置 notValid 属性，取消当前项的必填校验。
<template>
  <div class="mb-4">
    <rx-form
        ref="rxFormRef"
        :formProps="formProps"
        :form-items="formItems"
    />
    <rx-button @click="rxFormRef.validate()" type="primary" size="small">提交</rx-button>
  </div>
</template>
<script lang="ts" setup>
//内置的生成响应式表单的工具函数,返回一个代理对象,为了更好的支持TS请使用该函数生成配置项
import { useFormReactive , useFormItemReactive} from "@ruoxing/ruoxing-ui"
import {ref} from "vue";
const rxFormRef = ref()
// 对应el-form的props 如label-width等
const formProps = useFormReactive({
  labelWidth: '70px',
  labelPosition: 'left',
  //新增属性(控制每一个表单项/el-form-item的占位宽度)
  span: 24,
  //+设置所有项目为必填项
  allRequired: true,
})
// 表单项/el-form-item的生成配置
const formItems = useFormItemReactive([
  {
    // 对应el-form-item的props 如label等
    formItemProps:{
      label: "名称",
      prop: "username",
      //+自定义必填校验不通过的提示信息(未设置时，默认提示信息为“当前项不能为空!”)
      requireTipText: "用户名不能为空!"
    },
    //想要在el-form-item中渲染的组件名(el-input)/.vue文件/tsx语法
    component:'el-input',
    //需要传递给组件的props
    props:{
      //默认的v-model绑定值
      vModel:'',
      placeholder: "请输入用户名",
    }
  },
  {
    formItemProps:{
      label: "密码",
      prop: "password",
    },
    component:'el-input',
    props:{
      type: 'password',
      vModel:'',
      placeholder: "请输入密码",
      type: 'password'
    }
  },
  {
    formItemProps:{
      label: "邮箱",
      prop: "email",
      //+指定当前项不受formProps的allRequired控制
      notValid:true,
    },
    component:'el-input',
    props:{
      vModel:'',
      placeholder: "请输入邮箱",
    }
  },
])
</script>
  • 使用 formItemProps.defaultValidType 快速应用内置校验规则。

内置校验规则 ​

defaultValidType 支持以下校验规则：

  • email：校验邮箱格式
  • phone：校验手机号格式
  • idCard：校验身份证号格式
  • url：校验 URL 格式
  • isNumber：校验是否为数字
  • isChinese：校验是否为中文
  • isEnglish：校验是否为英文
  • password：校验密码格式
  • zipCode：校验邮政编码

    },
  },
])
</script>
<template>
  <div class="mb-4">
    <RxForm :form-items="formItems" :form-props="formProps"></RxForm>
  </div>
